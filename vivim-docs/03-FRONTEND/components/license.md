## Data Transparency

### What We Collect

| Data Type | Why | Retention |
|-----------|-----|-----------|
| Email | Account creation | Until deletion |
| Usage data | Product improvement | Aggregated, anonymous |
| Support tickets | Customer service | 2 years |
| Payment info | Billing | Payment processor only |

### What We DON'T Collect

- Conversation content (encrypted client-side)
- Encryption keys (you hold them)
- AI prompts or responses (we never see them)
- Data for training (we never train on user data)

### Your Rights

- **Access** — Download your data anytime
- **Export** — JSON, Markdown, or PDF formats
- **Delete** — Complete erasure on request
- **Portability** — Move to your own infrastructure

[Data Request →](/settings/data)

## Security Transparency

### Security Practices

| Practice | Status |
|----------|--------|
| Encryption at rest | AES-256 |
| Encryption in transit | TLS 1.3 |
| Zero-knowledge architecture | ✓ |
| Penetration testing | Annual |
| Bug bounty program | Active |
| SOC 2 certification | In progress |

### Vulnerability Disclosure

**Responsible disclosure policy:**

1. Report via security@vivim.app
2. We acknowledge within 24 hours
3. Fix timeline: Critical (24h), High (7d), Medium (30d)
4. Public disclosure after fix

**Bug bounty rewards:**

- Critical: $5,000 - $25,000
- High: $1,000 - $5,000
- Medium: $250 - $1,000

[Report a Vulnerability →](mailto:security@vivim.app)
