# For You - Recommendations

The **For You** feature uses AI to recommend relevant past conversations based on what you're doing. It's like having an intelligent assistant that remembers everything and shows you what matters.

---

## How It Works

When you visit For You, VIVIM:

1. Analyzes your conversation library
2. Considers your recent activity
3. Identifies relevant topics and themes
4. Ranks conversations by relevance
5. Presents recommendations with explanations

---

## Using For You

1. Navigate to **For You**
2. Scroll through recommendations
3. Tap any conversation to open
4. Swipe left to dismiss recommendations

---

## Recommendation Cards

Each recommendation shows:
- **Conversation title**
- **Why recommended** (topic match, related content)
- **Source** (which provider)
- **Match score**

---

## Customizing Recommendations

### Topic Filters
- Filter by specific topics
- See which topics VIVIM has detected
- Toggle topics on/off

### Recommendation Settings
Access via the **settings icon**:
- **Diversity** - Show more variety
- **Relevance** - Show most similar
- **Recency** - Prioritize recent
- **Reset dismissed** - Bring back dismissed items

---

## Dismissing Recommendations

### Individual
- Swipe left on a recommendation
- Tap the X button

### Reset All
- Go to settings
- Tap **Reset Dismissed**
- All previously dismissed items return

---

## Analytics

Track recommendation performance:
- Total recommendations shown
- Click-through rate
- Most common sources
- Recent activity

Access via **Analytics** page.

---

## Tips

1. **Visit regularly** - Recommendations improve as VIVIM learns
2. **Dismiss irrelevant** - Helps VIVIM learn preferences
3. **Explore topics** - Check what topics are detected
4. **Use filters** - Narrow down to relevant topics
