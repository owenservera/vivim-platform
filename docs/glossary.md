# Glossary

Complete reference for VIVIM terminology.

## Core terms

| Term | Definition |
|---|---|
| **ACU** | Atomic Context Unit — the smallest unit of context that retains standalone meaning |
| **Context Engine** | The 8-layer assembly pipeline that builds context bundles for AI providers |
| **DID** | Decentralized Identifier — W3C-compliant self-sovereign identity |
| **Open Core** | Open source foundation layer with commercial enterprise add-ons |
| **Sovereignty** | Full ownership and control of your AI memory data |
| **Zero-Knowledge** | Server cannot decrypt or access user data — only stores encrypted blobs |

## Memory types

| Type | Description | Example |
|---|---|---|
| **Episodic** | Specific events and conversations | "On March 3, we discussed deployment" |
| **Semantic** | General knowledge and facts | "We use Bun as the runtime" |
| **Procedural** | How-to knowledge and workflows | "Deploy via `bun run build` then push" |
| **Factual** | Concrete, verifiable facts | "The API runs on port 3000" |
| **Preference** | Personal likes, dislikes, habits | "Prefers TypeScript with strict mode" |
| **Identity** | Self-concept and roles | "Full-stack developer with blockchain experience" |
| **Relationship** | Social dynamics and interactions | "Team prefers async communication" |
| **Goal** | Objectives and targets | "Building decentralized AI memory" |
| **Project** | Project-specific context | "This repo uses Prisma with PostgreSQL" |

## Context layers (L0-L7)

| Layer | Name | Purpose | Token Budget |
|---|---|---|---|
| **L0** | Core Identity | Who you are | ~200 tokens |
| **L1** | Active Goals | Current objectives | ~400 tokens |
| **L2** | Relevant Memories | Pertinent facts and preferences | ~2,000 tokens |
| **L3** | Project Context | Current project state | ~1,500 tokens |
| **L4** | Relationship Context | Social dynamics | ~800 tokens |
| **L5** | Procedural Knowledge | How you work | ~1,000 tokens |
| **L6** | Episodic Context | Recent conversation history | ~3,000 tokens |
| **L7** | Strategic Context | Long-term patterns | ~1,000 tokens |

## Technical terms

| Term | Definition |
|---|---|
| **CRDT** | Conflict-free Replicated Data Type — enables conflict-free distributed sync |
| **DAG** | Directed Acyclic Graph — data structure for integrity verification |
| **MCP** | Model Context Protocol — open standard for AI tool data access |
| **Merkle Tree** | Cryptographic hash tree for data integrity verification |
| **Vector Embedding** | Numerical representation of text for semantic similarity search |
| **JIT Retrieval** | Just-in-time loading for unpredictable queries |
| **BM25** | Keyword-based full-text search algorithm |
| **libp2p** | Modular peer-to-peer networking stack |
| **ActivityPub** | Open federation protocol for decentralized social networks |
| **AES-256-GCM** | Authenticated encryption algorithm used by VIVIM |
| **Ed25519** | Digital signature algorithm for identity verification |
| **HKDF** | HMAC-based key derivation function |
