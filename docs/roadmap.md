# Roadmap

VIVIM's development roadmap across the open core and commercial layer.

## Open Core Roadmap

### Phase 1: Establish the Standard (Q1-Q2 2025)

| Pillar | Feature | Status |
|---|---|---|
| **ACU Specification** | Core ACU spec v1.0 | Live |
| **Provider Import** | OpenAI + Claude parsers | Live |
| **SDK** | SDK v1.0 (TypeScript) | Live |
| **MCP Server** | Claude Desktop, Cursor integration | Live |
| **Self-Hosting** | Docker Compose deployment | Live |

### Phase 2: Build the Ecosystem (Q3 2025)

| Pillar | Feature | Status |
|---|---|---|
| **Provider Import** | 5+ provider parsers | In Progress |
| **Integrations** | LangChain/LlamaIndex adapters | In Progress |
| **Automation** | n8n/Make/Zapier patterns | Planned |
| **Browser Extension** | Auto-capture from web UIs | Planned |
| **Self-Hosting** | Kubernetes Helm chart | Planned |

### Phase 3: Grow the Community (Q4 2025 - Q1 2026)

| Pillar | Feature | Status |
|---|---|---|
| **Plugins** | Obsidian/Notion/Logseq integrations | Planned |
| **Extensibility** | Community parser framework | Planned |
| **Storage** | S3-compatible storage backend | Planned |
| **Security** | Key rotation, enhanced encryption | Planned |
| **Community** | 10+ external contributors | Planned |

## Commercial Layer Roadmap

| Quarter | Feature | Description |
|---|---|---|
| **Q3 2025** | Cloud Beta | Managed VIVIM Cloud service |
| **Q4 2025** | Teams | Team memory sharing, admin panel |
| **Q1 2026** | SOC 2 | SOC 2 Type II compliance |
| **Q2 2026** | HIPAA | HIPAA BAA for healthcare |
| **Q2 2026** | Enterprise | SSO/SAML, RBAC, audit logs |
| **Q4 2026** | FedRAMP | Federal authorization |

## Current priorities

Right now, the team is focused on:

- **Provider import expansion** — Gemini, Grok, DeepSeek, Perplexity parsers
- **CRDT sync hardening** — Multi-circle team memory synchronization
- **SDK documentation** — Complete API reference and guides
- **MCP server polish** — Production-ready Claude Desktop integration
