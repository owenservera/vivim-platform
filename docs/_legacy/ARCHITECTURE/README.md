## How to Use

### For New Developers

1. Start with `01-tech-stack.md` to understand the technology choices
2. Read `02-architecture-overview.md` for system understanding
3. Reference `03-api-endpoints.md` for API usage

### For Architects

1. Review all core documents
2. Check corresponding zcompanies files for gaps
3. Review `zcompanies-gap-analysis.md` for priority items

### For Operations

1. Focus on `02-architecture-overview.md` for deployment
2. Review data flows in `04-data-flows.md`
3. Check gap analysis for monitoring gaps
