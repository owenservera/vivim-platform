## Technical Differentiation

### Not a Wrapper — Real Infrastructure

This isn't a simple UI on top of APIs. VIVIM is a sophisticated distributed system:

| Layer | Technology | Why It Matters |
|-------|------------|----------------|
| **Capture** | Playwright + Stealth Automation | Headless browser automation to extract conversations from any AI provider — works even with authentication |
| **Processing** | ACU (Atomic Chat Unit) Segmentation | Conversations decomposed into granular, embeddable knowledge pieces with quality scoring |
| **Search** | pgvector + Hybrid Retrieval | Semantic search across embeddings, with lexical fallback |
| **Context** | 8-Layer Personalization Engine | Identity, preferences, topics, entities, memories, JIT retrieval — adaptive token budgeting |
| **Storage** | Isolated Per-User Databases | Each user gets their own SQLite instance — complete data isolation |
| **Sync** | CRDTs (Yjs) + LibP2P | Conflict-free synchronization across devices, P2P-ready |
| **Identity** | DID + Ed25519 Signatures | Self-sovereign identity, cryptographic authorship verification |
| **Privacy** | Quantum-Resistant Crypto | ML-KEM-1024 (Kyber) key exchange, XSalsa20-Poly1305 encryption |
| **Sharing** | Policy-Based Access Control | Granular permissions with cryptographic verification |
| **Storage** | On-Chain Anchoring | Optional IPFS + blockchain proof of existence |

### Extensible Architecture
Modular node-based design. Plugin system. Build autonomous agents on top.

## Code Example

```typescript
import { VivimSDK } from '@vivim/sdk';

const sdk = new VivimSDK({
  identity: { did: 'my-node-' + Math.random() },
  storage: { encryption: true }
});

await sdk.initialize();

// Capture a conversation
await sdk.capture.fromProvider('chatgpt', sessionToken);

// Search semantically
const results = await sdk.search.semantic('vector database optimization');

// Share with granular permissions
await sdk.sharing.share(acuId, {
  recipients: ['did:key:alice...'],
  permissions: { canView: true, canRemix: true }
});
```
